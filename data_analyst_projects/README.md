# Data Analyst Portfolio Projects (CSV + SQL + Python + Excel)

This package contains **5 complete, job-ready data analyst projects** aligned to your dashboard visuals.

## Projects Included
1. **01 Minnesota Interstate Traffic Volume** — traffic patterns by month/hour/weather/holiday  
2. **02 Netflix Content Strategy** — catalog mix, ratings, genres, country distribution, additions over time  
3. **03 Executive Overview - Profitability** — KPIs, profit ratio by state, monthly trends by segment/category  
4. **04 U.S. Baby Names Trends** — top names by gender, era peaks, trend queries  
5. **05 Athens Airbnb Pricing & Supply** — average price by neighborhood + supply concentration

## What each project includes
- `data/raw/*.csv` and `data/clean/*.csv`
- `python/*.py` (cleaning + analysis)
- `sql/*.sql` (create tables + analysis queries)
- `excel/*.xlsx` (formatted dashboard workbook)
- `assets/` (your screenshot reference; some projects include PBIX)

## Quick Start
1. Open any project folder.
2. Use `python/01_clean_data.py` to reproduce the cleaned dataset.
3. Load `data/clean/*.csv` into Power BI / Tableau.
4. Run queries in `sql/02_analysis_queries.sql` in your SQL tool.
5. Open the Excel dashboard in `excel/`.

> Tip: In Power BI, connect to **cleaned CSV** and build visuals that match your screenshots.
